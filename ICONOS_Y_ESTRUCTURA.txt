1. ¿A QUÉ SE DEDICA EL PROYECTO?
----------------------------------------------------------------
Aplicación web de FINANZAS PERSONALES. Permite a un usuario
rastrear ingresos, gastos, presupuestos y metas de ahorro, con
dashboard, gráficos, categorías personalizadas y multi-idioma.

  - Registro/seguimiento de INGRESOS y GASTOS por categoría
  - PRESUPUESTOS mensuales y METAS de ahorro
  - Dashboard con gráficos (donut, sparkline, cashflow, heatmap)
  - Categorías personalizadas y favoritas por usuario
  - Multi-moneda (PEN / USD / EUR) y multi-idioma (ES / EN)
  - Autenticación email/password con Supabase
  - Desplegada en: https://www.jeshu.cfd

  STACK:
    Framework ... Next.js 16.2.6 (App Router)
    UI .......... Material UI (MUI) v9
    Auth + DB ... Supabase (email/password)
    Fechas ...... MUI X Date Pickers + dayjs
    Estado ...... React Context + localStorage
    Lenguaje .... TypeScript (rutas/config) + JSX (componentes)
    Deploy ...... Vercel

3. ICONOS DEL PROYECTO
----------------------------------------------------------------
Librería: @mui/icons-material (MUI v9). SVG, sin emojis ni
assets propios. 88 iconos únicos. Variante: Rounded
(Material You / Material 3) desde v1.2.0.

El sistema tiene TRES piezas:

  (A) MÓDULO CENTRAL de iconos ... src/theme/icons.js
        Re-exporta los 88 iconos en variante Rounded (deep-import,
        mejor tree-shaking). Los 14 componentes/páginas importan
        desde aquí, NO del barrel @mui/icons-material. Cambiar de
        estilo a futuro (p.ej. Outlined) = editar solo este archivo.
        Casos especiales:
          GitHub / Google / YouTube -> sin variante Rounded (Filled).
          ErrorOutlined             -> mapea a ErrorRounded.
  (B) MAPEO categoría → icono .... src/components/AddTransactionModal.jsx
  (C) ANIMACIÓN centralizada ..... src/theme/materialTheme.js

3.C  ANIMACIÓN DE ICONOS  (theme/materialTheme.js:45-88)
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
El theme MUI centraliza el comportamiento animado (no la lista):
  - MuiSvgIcon ............ transición suave de transform y color
  - MuiIconButton ........ hover scale(1.12), click scale(0.92)
  - MuiFab ............... hover translateY(-2px) scale(1.06)
  - MuiTab ............... icono scale(1.18) en hover + animación
                          "iconPop" al seleccionar
  - MuiBottomNavigationAction ... mismo pop al seleccionar
  - MuiCard (avatar) ..... "pop" sutil al hover de la tarjeta
  - Respeta prefers-reduced-motion (guard en globals.css)


4. RESUMEN RÁPIDO
----------------------------------------------------------------
  - 88 iconos SVG únicos de @mui/icons-material, variante Rounded
    (Material You), centralizados en src/theme/icons.js.
  - Categorías -> icono se ASIGNA en AddTransactionModal.jsx
    (EXPENSE_ICONS / INCOME_ICONS).
  - Iconos de UI viven en cada componente (tabs, auth, settings).
  - La ANIMACIÓN de todos los iconos se controla desde el theme.
  - Proyecto = app de finanzas personales (Next.js + MUI + Supabase).
